#  Phishing Awareness Training 🛡️

An interactive web-based security awareness training module built as part of the **CodeAlpha Cybersecurity Internship** (Task 2).

---

## 📌 What It Does

PhishGuard teaches users how to recognize and avoid phishing attacks through a guided, slide-based learning experience with a built-in quiz. No installation needed — just open the HTML file in any browser.

---

## 🎯 Features

- 🎞️ **Slide-based module** — 6 interactive slides navigated with Previous / Next buttons and dot indicators
- 📧 **Email red flags** — a realistic fake phishing email with annotated warning signs
- 🌐 **Fake website detection** — side-by-side URL comparisons showing real vs. fake domains
- 🧠 **Social engineering tactics** — explains how attackers use fear, urgency, authority, and sympathy
- ✅ **Best practices** — actionable tips to stay safe online
- 🔍 **Real-world examples** — case studies of actual phishing attacks
- 🧪 **Interactive quiz** — 5 multiple choice questions with instant feedback and auto-advance
- 📱 **Responsive design** — works on desktop and mobile screens
- 🎨 **Dark cybersecurity theme** — professional UI with cyan accents and smooth animations

---

## 📚 Module Breakdown

| Slide | Topic | What You Learn |
|---|---|---|
| 1 | Hero / Landing | Overview of the module and what phishing is |
| 2 | Phishing Emails | 4 red flags: mismatched sender, urgency, generic greeting, suspicious links |
| 3 | Fake Websites | How to read URLs correctly, why HTTPS alone isn't enough |
| 4 | Social Engineering | Psychological tactics: fear, authority, greed, sympathy |
| 5 | Best Practices | How to protect yourself with 4 key habits |
| 6 | Quiz | 5 questions to test everything you learned |

---

## 🧪 Quiz Topics Covered

- What to do when you receive a suspicious email from a known brand
- How to identify a fake sender address
- Common social engineering tactics
- What a padlock / HTTPS actually means
- How to spot a look-alike URL (e.g. `paypaI.com` vs `paypal.com`)

---

## 🚀 How to Run It

No server, no dependencies, no installation needed.

1. Download `phishing_awareness_interactive.html`
2. Double-click the file to open it in your browser
3. Click **Start Learning** and navigate through the slides
4. Click **Take Quiz** or navigate to the last slide to test yourself

> Works in Chrome, Firefox, Edge, and Safari.
